# Основи ООП на Python

Цей репозиторій містить практичні завдання з курсу об'єктно-орієнтованого програмування.

## Структура проекту

- `car.py` — Клас «Автомобіль» (модель, рік, виробник, об'єм двигуна, колір, ціна).
- `book.py` — Клас «Книга» (назва, автор, рік, видавець, жанр, ціна).
- `stadium.py` — Клас «Стадіон» (назва, дата відкриття, країна, місто, місткість).

## Як запустити

Кожен файл можна запустити окремо для тестування функціоналу:
```bash
python car.py
```
